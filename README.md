# NextWave Design Studio

AI-powered design workspace for generating and visualizing interior layouts in real-time.

## 🚀 Features
- AI-generated design layouts
- Interactive 2D canvas engine
- Workspace UI (Figma-style)
- Real-time design rendering

## 🛠 Tech Stack
- Next.js
- React
- Tailwind CSS
- Vercel

## 📦 Installation

```bash
npm install
npm run dev
